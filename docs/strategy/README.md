## 🧭 Cara Pakai untuk Hackathon Submission

Saat presentasi ke judges, urutan storytelling yang direkomendasikan:

1. **"Siapa user kita"** — buka dengan Pak Bambang dan Bu Sari (B2G + B2B = bisnis nyata)
2. **"Apa pain mereka"** — sebut spesifik (chatbot existing resolution < 40 %, vendor asing tidak compliance)
3. **"Bagaimana mereka jadi pelanggan"** — show customer journey + cycle time
4. **"Bagaimana kita berbeda"** — compliance + bahasa daerah + on-prem (Trust & Compliance pilar)
5. **"Berapa kita charge"** — pricing land-and-expand
6. **"Berapa besar pasarnya"** — TAM/SAM/SOM tertulis di PERSONAS

Total slot bicara: ~5-7 menit untuk seluruh strategi.
